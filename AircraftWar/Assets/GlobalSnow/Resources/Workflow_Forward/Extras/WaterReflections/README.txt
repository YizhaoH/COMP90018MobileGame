Water.cs replacement for Unity Water Pro system.

This script adds snow reflection support to water.
It replaces Water.cs located in Standard Assets/Environment/Water/Water/Scripts folder.

To use it:

1) Uncompress Water.cs.zip archive (double click on it in Unity Project explorer to uncompress it)
2) Replace existing Water.cs file in Standard Assets with this version.
3) Select your camera and exclude Water layer from Layer Mask (if you have not done it already).
